# mongoose-web-server

a web server written on C using the mongoose http library.

## building
unix:
```sh
make
```

msvc (windows):
```cmd
msbuild
```

## running
unix:
```sh
make run
```

msvc (windows):
```cmd
msbuild /p:Run=true
```

## testing
unix:
```sh
make test
```

windows (msvc): test suite on windows not supported at this time.
